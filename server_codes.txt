
 СЕРВЕР АВТОРИЗАЦИИ

Коды сообщений клиента:

1001 - Авторизация по логину (login + password)
1002 - Авторизация по ключу (key)
1003 - Подписанная ключевая фраза (key + key phrase)

Коды сообщений сервера:

1001 - Новый игрок (key + secret)
1002 - Вход по логину успешен (key + secret)
1003 - Проверочная фраза (key phrase)
1004 - Вход по ключу успешен

Коды ошибок сервера:

1101 - Ошибка протокола
1102 - Неверный пароль
1103 - Неверная ключевая фраза
