Игровой сервер
 - точка входа
  - бой с последующим выходом
    из редиса берется по токену тип входа [2], ключ игрока и ключ боя
    из редиса получается по ключу профиль игрока
    комната с классом Game
  - регион с выходом в бой и обратно, и выходом с сервера
    из редиса берется по токену тип входа [1], ключ игрока, код региона и код точки входа
    из редиса получается по ключу профиль игрока
     периодически проверяется необходимость добавления пары и добавляется пара
     при разрыве дистанции в паре - пара разрывается
     при изменении направления или броске кошки пересчитывается столкновение по парам
     при достижении необнуленного столкновения - бой
      создание комнаты для боя с ограничениями по идентификатору
      уведомление игроков с номером комнаты
 - бой
   при входе обоих соперников в комнату инициализируется локация и эмуляция
